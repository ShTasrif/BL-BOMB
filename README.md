# BL-BOMB
A Powerful Banglalink SMS Bomber Tools
## Install on Terminal :

```

apt update -y && apt upgrade -y 

apt install python -y

apt install python3 -y

apt install git -y

apt install pip -y

apt install pip2 -y

pip install requests -y

pip install mechanize -y

git clone https://github.com/ShTasrif/BL-BOMB

```
## Run 
```
cd BL-BOMB
python3 bomber.py
```
